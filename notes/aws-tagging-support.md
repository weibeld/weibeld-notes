title: AWS Tagging Support
---

Support for assigning tags at the time the resource is created:

| Resource           | Tagging support |
| ------------------ | --------------- |
| EC2 Instance       | Yes             |
| Key Pair           | Yes             |
| Elastic IP Address | No              |
| Internet Gateway   | No              |
| Route Table        | No              |
| Security Group     | No              |
| Subnet             | No              |
| VPC                | No              |
